# 1. Aim

To develop a Python-based Expense Tracker using file handling and JSON.

# 2. Dataset

The dataset consists of expense records stored in JSON format with fields:

 * Amount
 * Category
 * Note
# 3. Source

Dataset was taken from Kaggle and converted to JSON

# 4. Working
 * User adds expenses
 * Data stored in JSON file
 * Program calculates totals and categories
