## Theory

### File Handling in Python

File handling allows us to store data permanently in files. Python provides built-in functions like:

* `open()` – to open a file
* `read()` – to read data
* `write()` – to write data
* Modes like:

  * `"r"` → Read
  * `"w"` → Write
  * `"a"` → Append

### Exception Handling

Exception handling is used to handle runtime errors and prevent program crashes.
Key components:

* `try` → Block where error may occur
* `except` → Handles the error
* `finally` → Executes always (optional)

Types used:

* `ZeroDivisionError`
* `ValueError`
* `FileNotFoundError`
* Custom exceptions
