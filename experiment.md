# Experiment 8: GUI and Backend Connectivity Using Tkinter and SQLite

## 1. Aim

To design GUI-based applications using Tkinter and connect them with backend databases using SQLite.

---

## 2. Theory

### Tkinter (GUI Library)

Tkinter is Python’s standard GUI (Graphical User Interface) library. It allows us to create windows, buttons, labels, text fields, etc.

Key components:

* `Tk()` → Main window
* `Label` → Display text
* `Entry` → Input field
* `Button` → Perform actions
* `Frame` → Organize layout

---

### SQLite (Database)

SQLite is a lightweight database stored in a file. It is used to:

* Store data permanenantly

* Store data permanently
* Perform operations like INSERT, DELETE, SELECT

Key functions:

* `connect()` → Connect to database
* `cursor()` → Execute SQL commands
* `commit()` → Save changes
